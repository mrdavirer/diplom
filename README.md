Обязательные системные требования

Node.js 16.0.0 или выше

npm 7.0.0 или выше

MySQL 5.7 или выше


Для запуска проект требуется установка зависимостей

Подготовка к запуску проекта
Шаг 1
Установить проект с GitHub
Шаг 2
Открыть скаченный проект в VisualStudioCode
Шаг 3
Открыть "Терминал" прописать 'cd frontend', после прописать 'npm i'
Шаг 4
Открыть "Терминал" прописать 'cd backend, main' после прописать 'go run main.go'

Запуск проект (если все для него уже было сделано и все команды прописаны)
Шаг 1
В терминале бэкэнда прописываем npm run dev
Шаг 2
В терминале бэкэнда прописываем npm run dev

После этого проект запущен осталось открыть его в браузере
Api http://localhost:3001
Frontend (Localhost) http://localhost:5173/

Тестовые аккаунты:
   Админ: admin / password
   Пользователь: user1 / password

А также для проерки входа и регистрации, можно создать своего пользователя
